DFI — библиотека готовых функций для программного формирования интерфейса форм 1С.
Она помогает быстрее создавать элементы формы, реквизиты, команды и типовые структуры свойств без ручного описания однотипного кода.

## Установка
Для подключения библиотеки скопируйте код из `ModuleDFI.bsl` в новый общий модуль конфигурации, например `ДФИ`.

## Возможности
1. Создание элементов формы. Имена функций соответствуют пунктам меню интерактивного создания элементов формы и начинаются с префикса `Сзд`, например `ДФИ.СздГруппаОбычнаяБезОтображения()`.
2. Создание реквизитов и команд. В модуле есть функции для добавления реквизитов формы, реквизитов таблицы и команд.
3. Описание параметров. Функции сопровождаются комментариями с назначением параметров и ожидаемыми типами значений.
4. Готовые структуры свойств. Библиотека содержит вспомогательные функции для подготовки типовых свойств элементов формы.
5. Специфические элементы. Доступны элементы, которые сложнее настроить интерактивно, например гиперссылка, выпадающий список и динамический список.
6. Удаление элементов и реквизитов. Предусмотрены функции для удаления созданных реквизитов, элементов формы и связанных пар «реквизит + элемент».

## Описание элементов
### Группы и страницы
- `СздГруппаОбычнаяБезОтображения()` — создает обычную группу без визуального оформления и заголовка. Удобна как технический контейнер для компоновки элементов.
- `СздГруппаОбычная()` — создает обычную группу с заголовком и слабым выделением. Подходит для визуального объединения связанных элементов формы.
- `СздГруппаСтраниц()` — создает группу страниц. Используется как контейнер для вкладок и может принимать обработчик события смены страницы.
- `СздСтраница()` — создает страницу внутри группы страниц. Позволяет задать заголовок, группировку и параметры растягивания.
- `СздГруппаКоманднаяПанель()` — создает группу вида «Командная панель». Используется для размещения команд и кнопок.
- `СздГруппа()` — универсальная функция создания группы формы с настройками через структуру свойств.

### Поля
- `СздПоле()` — создает поле формы одного из популярных видов: поле ввода, флажок, индикатор, переключатель, период или HTML-документ.
- `СздПолеФлажок()` — создает поле флажка с выбором варианта отображения: флажок, тумблер или выключатель.
- `СздПолеГиперссылка()` — создает поле надписи в режиме гиперссылки. По умолчанию скрывает отдельный заголовок элемента.
- `СздПолеНадписи()` — создает поле надписи для вывода текста или значения реквизита без режима редактирования.

### Декорации и кнопки
- `СздДекорация()` — универсальная функция создания декорации формы с настройкой свойств и событий.
- `СздДекорацияНадпись()` — быстро создает декорацию-надпись с заданным текстом и возможностью растянуть ее по горизонтали.
- `СздДекорацияКартинка()` — быстро создает декорацию-картинку из библиотеки картинок конфигурации.
- `СздКнопка()` — создает кнопку формы, связывает ее с командой и позволяет выбрать вид кнопки: обычная кнопка или гиперссылка.

### Таблицы и списки
- `СздДинамическийСписок()` — создает реквизит динамического списка и связанную с ним таблицу формы. Поддерживает произвольный запрос, параметры запроса, основную таблицу и настройку колонок.
- `СздТаблицаФормы()` — создает таблицу формы, привязанную к указанному пути данных. Может автоматически создать колонки и отдельную командную панель.
- `СздКолонкаТаблицыФормы()` — создает колонку таблицы формы. Поддерживает поле ввода, поле надписи, поле флажка и поле картинки.
- `СздГруппаКолонокТаблицыФормы()` — создает группу колонок таблицы формы с настройкой группировки, заголовка и отображения в шапке.

### Специфические элементы
- `СздВыпадающийСписок()` — создает строковый реквизит и связанное поле ввода с кнопкой выпадающего списка. Значения выбора передаются массивом.
- `СздПереключатель()` — создает числовой реквизит и связанное поле переключателя или тумблера. Значения выбора передаются массивом.
